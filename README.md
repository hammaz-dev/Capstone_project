# Online Learning Platform
A modern, responsive landing page built with HTML and CSS for an Online Learning Platform. It includes a hero section, categories, courses, instructors, testimonials, blogs, FAQs, and a newsletter signup. Designed using Flexbox, Grid, and Google Fonts to ensure clean UI/UX and full mobile responsiveness. Easy to expand into a complete web application.
Tech Used: HTML5, CSS3
Focus: Responsive design, reusable components, clean layout.
